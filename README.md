## About Me

I'm a professional developer (backend, machine learning, data science). An avid reader, writer, runner, hiker, rock climber. A polymath of varied interests: history (especially ancient history), political science, physics, astronomy, computer science, information theory (informatics), computational linguistics, anthropology, and paleontology. I've thru-hiked the Appalachian Trail (class of 2017). I've published in academic journals.
